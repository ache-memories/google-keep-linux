# 📝 Google Keep Desktop for Linux

A lightweight, native, and privacy-focused tool built for Linux.

## 📸 Screenshot
<p align='center'><img src='https://www.gstatic.com/images/branding/product/2x/keep_96dp.png' width='64' height='64'></p>

## ✨ Features
- 🚀 **Ultrafast Launch:** Opens in milliseconds.
- 💾 **Low RAM Usage:** Uses native WebKit engine.
- 🛡️ **Privacy Centric:** No trackers.

## 🛠️ Installation
```bash
git clone https://github.com/ache-memories/google-keep-linux.git
cd google-keep-linux
./install.sh
```
