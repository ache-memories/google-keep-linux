# 📝 Google Keep Desktop for Linux

A lightweight, native, and privacy-focused tool built for Linux.

## ✨ Features
- 🚀 Ultrafast Launch
- 💾 Low RAM Usage
- 🛡️ Privacy Centric

## 🛠️ Installation
```bash
git clone https://github.com/ache-memories/google-keep-linux.git
cd google-keep-linux
./install.sh
```
